<h1>Spend Track</h1>
Welcome to Spend Track! This full-stack web application helps you efficiently track your expenses. Utilizing React with Vite in the frontend and Supabase for the backend, Spend Track implements user authentication and CRUD operations seamlessly, offering a responsive design for a smooth user experience.

<h2>Features</h2>
User Authentication: Secure authentication ensures data privacy.
CRUD Operations: Manage expenses with Create, Read, Update, and Delete functionalities.
Efficient Management: Categorize transactions and monitor spending habits.
Responsive Design: Enjoy a consistent experience across devices.

<h2>Technologies Used</h2>
Frontend: HTML, CSS, Javascript, React, Vite
Backend: Supabase
Visualization: Chart.js (for visualizing expense data)

Deployment
Deploy Spend Track to hosting platform Vercel.
